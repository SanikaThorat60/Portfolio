# Premium Developer Portfolio Template

Welcome to your new premium portfolio template! This project is built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**, designed to give you a high-end, highly-responsive online presence with zero hassle.

## 🚀 Key Features
- **Zero React Knowledge Required**: Edit your entire portfolio from a single `config.jsx` file.
- **Dark/Light Mode**: Seamless, system-aware theme toggling.
- **Fully Responsive**: Flawless design on mobile, tablet, and desktop.
- **Smooth Animations**: High-performance scroll animations via Framer Motion.

## 📦 Quick Start Guide

### 1. Install Dependencies
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.
Open your terminal in the project directory and run:
```bash
npm install
```

### 2. Start the Development Server
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to see the portfolio!

## 🎨 How to Customize (The Easy Way)

You do **not** need to touch any of the UI components to personalize this template.

1. Open `src/config.jsx`.
2. Replace all the placeholder text (like `[Your Name]`) with your actual information.
3. Replace the placeholder image URLs with links to your own project screenshots.
4. Save the file, and your entire portfolio will update automatically!

### Changing the Theme Color
The template currently features a striking "Yellow & Black" aesthetic. If you want to change the primary accent color:
1. Open `src/index.css`.
2. Locate the `--accent-color` and `--accent-glow` variables.
3. Replace the hex codes and RGB values with your preferred brand color.

## 🚀 Deployment

This template is optimized for easy deployment on free platforms like **Vercel** or **Netlify**.

1. Create a GitHub repository and push this code to it.
2. Log into [Vercel](https://vercel.com/) or [Netlify](https://www.netlify.com/).
3. Click "Import Project" and select your new repository.
4. The platform will automatically detect Vite and build your site. 
5. Click Deploy!
